# 站点结构与页面规则（纯导航版）

## 1. 页面结构
- 单页结构：`index.html`。
- 模块顺序：标题区 -> 搜索框 -> 分类筛选 -> 工具卡片网格。
- 不创建文章页、教程页、对比长文页。

## 2. 工具卡片字段
- 工具名
- 分类
- 一句话描述（不超过 24 字）
- 外链按钮：`Visit`

## 3. 交互规则
- 搜索：按工具名和描述实时过滤。
- 分类：点击分类按钮过滤。
- 外链：新窗口跳转（`target="_blank"`）。

## 4. 视觉规则
- 卡片等高，3列网格（移动端自动单列）。
- 按钮统一文案：`Visit`。
- 页面保持轻量，不加复杂动画和弹窗。

## 5. 维护规则
- 每周巡检一次失效链接。
- 每次只新增真实使用过或被广泛使用的工具。
- 删除长期不可访问或停止维护的工具。
